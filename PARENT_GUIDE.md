### Getting Started (First Time)

1. **Set it up together** - Let your child watch as you start the app
2. **Explore the dashboard** - Show them the two subjects (Math and Reading)
3. **Start with Level 1** - Build confidence with easier questions

- Alternate between Math and Reading
- Complete 2-3 quizzes
- Set a point goal to work toward

**Option 3: Weekend Challenge (30-45 minutes)**
- Work toward unlocking the next level
- Mix different difficulty levels
- Review areas that need more practice

### Do's âœ…

- **Praise effort, not just correct answers** - "Great thinking!" vs. "You got it right!"
- **Review explanations together** - Even for correct answers
- **Set achievable goals** - "Let's earn 50 points today!"

### Don'ts âŒ

- **Don't rush** - It's better to understand 5 questions than speed through 20

## ğŸ” Identifying & Addressing Struggles

### If They're Struggling with Math:
- Start with Level 1, even if unlocked higher
- Work through one question type at a time
- Use physical objects to visualize problems
- Consider adding more basic questions to `questions.json`

### If They're Struggling with Reading:
- Read passages aloud together
- Discuss the story before answering
- Teach them to highlight key words
- Practice finding answers in the text

### If They're Getting Frustrated:
- Take a break - learning should be fun!
- Go back to an easier level
- Celebrate small wins
- Remind them that mistakes = learning

## â“ FAQs

**Q: How long should practice sessions be?**
A: 10-15 minutes daily is better than 60 minutes once a week. Consistency matters more than duration.

**Q: My child keeps getting the same types of questions wrong. What should I do?**
A: Add more questions of that type to `questions.json`. Focus practice on that specific skill.

**Q: Should I let them retry questions they got wrong?**
A: Yes! The app currently shows explanations. You can add the same question with slight variations for extra practice.

**Q: What if they lose interest?**
A: Take a break, try a different subject, or add questions about topics they enjoy. Make it relevant to their interests.

**Q: How do I know if they're ready for the test?**
A: Consistent 80%+ accuracy on Level 2-3 questions shows good preparation. Focus on building confidence!
